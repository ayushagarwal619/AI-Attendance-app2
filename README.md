# 🤖 AI Attendance App

An AI-powered attendance management system built using Streamlit, Face Recognition, and Supabase. The application automates attendance tracking by recognizing faces and securely storing attendance records in a cloud database.

## 🚀 Features

- Face Recognition based attendance marking
- Real-time attendance tracking
- Secure cloud database integration using Supabase
- User-friendly Streamlit interface
- Automated record management
- Fast and efficient attendance processing

## 🛠️ Tech Stack

- Python
- Streamlit
- Face Recognition
- Dlib
- Scikit-Learn
- NumPy
- Pandas
- Supabase
- Pillow

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/your-username/AI-Attendance-app.git
cd AI-Attendance-app
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

Activate virtual environment:

**Windows**

```bash
venv\Scripts\activate
```

**Linux / Mac**

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run Application

```bash
streamlit run app.py
```

## 📊 Workflow

1. User opens the application.
2. Face image is captured/uploaded.
3. AI model detects and recognizes faces.
4. Attendance is marked automatically.
5. Records are stored securely in Supabase.

## 🔒 Database

This project uses Supabase for:

- User management
- Attendance storage
- Secure cloud synchronization
